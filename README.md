## 📖 Elevator Pitch

**TARIK** is an innovative, gamified DeFi platform where users can stake their native coins (MON) into one of two competing factions (e.g., Bull vs. Bear, $DEGEN vs. $HIGHER).

**The Golden Rules of TARIK:**
- 🛡️ **100% Principal Protection:** Your initial deposit is completely safe and can be withdrawn at any time after the round ends.
- 📈 **Yield Generation:** All staked funds simulate yield generation (similar to Aave or Compound).
- 🏆 **The Winning Side** gets their full principal back **+ ALL THE YIELD** generated by both sides.
- 💀 **The Losing Side** gets their full principal back (0% yield).

*Zero risk of losing your principal. Pure adrenaline and real yield.*

## ✨ Core Features (The Degen Angle)

TARIK isn't just another financial tool; we inject highly addictive GameFi elements to maximize user retention and deliver that dopamine hit.

### 📦 1. Victory Crates (Lootbox/Gacha Mechanics)
When your side wins, you don't just receive your yield as a boring number. You earn a **Victory Crate NFT**.
Crack open this crate (powered by stunning *Framer Motion* animations) to reveal your prize:
- **90% Chance:** Your normal yield share.
- **9% Chance:** Yield + Exclusive Tokens/Memecoins.
- **1% Chance:** Jackpot (SSR NFT, Special Badges, or Yield Multipliers).

*Experience the thrill of a gacha roll without ever spending a dime on the ticket!*

### ⚔️ 2. Dynamic "Tug-of-War" Arena
Our frontend features a real-time "Tug-of-War" arena. The indicator bar smoothly and dynamically shifts in response to the Total Value Locked (TVL) of each side. The battle of capital is visualized elegantly!

## 💻 Local Setup Guide

Follow these steps to run TARIK locally on your machine.

### Frontend Setup
```bash
cd fe
npm install

# Configure environment variables
cp .env.example .env.local

# Run the development server
npm run dev
```
Open `http://localhost:3000` in your browser.

### Smart Contract Setup (Backend)
If you wish to modify or redeploy the smart contracts to a testnet or local node:
```bash
cd sc
forge build
forge test

# Deploy to Monad Testnet (Ensure your PRIVATE_KEY is set in the .env file)
forge script script/DeployFresh.s.sol:DeployFresh --rpc-url https://testnet-rpc.monad.xyz --broadcast --legacy
```
